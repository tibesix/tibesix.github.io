CSS Implementatie op Verschillende Websites

1. Vives.be

  • Link naar aparte CSS-bestanden: Vives.be gebruikt voornamelijk <link> verwijzingen naar externe CSS-bestanden om stijlen te laden.
  • Inline style attributen: Er wordt minimaal gebruikgemaakt van inline styles voor specifieke aanpassingen.
  • Style-blokken: Geen uitgebreide <style> blokken in de HTML.

2. Miras.be

  • Link naar aparte CSS-bestanden: Miras.be gebruikt ook externe CSS-bestanden via <link> verwijzingen.
  • Inline style attributen: Beperkt gebruik van inline styles voor specifieke elementen.
  • Style-blokken: Geen grote <style> blokken in de HTML gevonden.

3. VRT NWS (vrt.be/vrtnws)

  • Link naar aparte CSS-bestanden: VRT NWS maakt voornamelijk gebruik van <link> verwijzingen naar externe CSS-bestanden.
  • Inline style attributen: Inline styles worden mogelijk gebruikt voor specifieke dynamische content, maar zijn over het algemeen minimaal.
  • Style-blokken: Waarschijnlijk geen grote <style> blokken in de HTML structuur.

Opdracht: CSS en Tekstcentrering

1. Waarom staat de tekst in de blokken horizontaal gecentreerd?

De tekst staat horizontaal gecentreerd omdat in de CSS is aangegeven: 

div {
    text-align: center;
}

Dit zorgt ervoor dat alle tekst in <div>-elementen horizontaal gecentreerd wordt.

2. Waarom heeft vertical-align: center geen effect?

vertical-align werkt alleen op inline, inline-block, of tabelcel-elementen. De .area-blokken gebruiken display: inline-block, maar hun tekst wordt niet beïnvloed door deze property.

3. Waarom zorgt vertical-align: middle niet voor verticale centrering?

vertical-align: middle positioneert elementen ten opzichte van andere inline of inline-block elementen, maar het centreert de tekst niet binnen een block-element.

4. Wat doet vertical-align wel?

vertical-align bepaalt de uitlijning van inline- of inline-block-elementen ten opzichte van elkaar. Bijvoorbeeld voor afbeeldingen naast tekst.

5. Hoe centreer je één enkele regel tekst verticaal?

Gebruik de line-height die gelijk is aan de hoogte van het element: 

.area {
    line-height: 100px; /* Gelijk aan de hoogte */
    text-align: center;
}

6. Hoe centreer je meerdere regels tekst verticaal?

Gebruik display: table-cell en vertical-align: middle: 

.area {
    display: table-cell;
    vertical-align: middle;
    text-align: center;
}

7. Waarom krijgt de regel voor .tall voorrang?

De CSS-selector voor .tall is specifieker dan de algemene selector .area. Specifiekere regels krijgen volgens de CSS-specificiteitsregels voorrang.

Test bijvoorbeeld deze regel, die nog specifieker is: 

.area.tall {
    height: 150px;
}
